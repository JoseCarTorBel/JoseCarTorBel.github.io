Trabajos realizados

Test Nvidia Jetson TX2

Realización de tests de eficiencia energética sobre plataformas de bajo consumo como Nvidia Jetson TX2. Pruebas realizadas con CUDA y redes neuronales como ImageNet, AlexNet o GoogleNet para el reconocimiento de imágenes.

Creación de Redes Neuronales para el Reconocimiento de Audio

Creación de redes neuronales de tipo VGG y ResNet para el reconomiento de sonidos urbanos. El proyecto incluye la búsqueda de hiperparámetros y el estudio de la eficiencia energética y temporal para dispositivos de bajo consumo como Google Coral Dev Board, Nvidia Jetson TX2 ó Jetson AGX Xavier.

Redes LoRa para la monitorización de entornos industriales

Redes LoRa para monitorizar valores ambientales y del entorno como temperatura, humedad o nivel de gases. El proyecto se realiza mediante el uso de ESP32 LoRa TTGO, con los que creamos una red mallada de tipo LoRa a nivel físico. La red se comunica con un servidor para almacenar los datos mediante el protocolo de cola MQTT y se guardan los datos en una BBDD InfluxDB.
